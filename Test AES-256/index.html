🛡️ Security Information

AES-256-CBC Encryption

  • Uses 256-bit keys for maximum security
  • CBC mode with random IV for each encryption
  • PKCS7 padding for proper block alignment
  • Authenticated encryption (Encrypt-then-MAC)

Security Warnings

  • 🔑 Keep your keys secure and never share them
  • 🔄 Never reuse the same IV with the same key
  • ✅ Always verify HMAC before decryption
  • 🚫 Don't transmit keys over insecure channels
  • 💾 Keys are stored in memory only (not saved)

Output Format

Base64(IV + Ciphertext + HMAC-SHA256)

  • IV: 16 bytes (random for each encryption)
  • Ciphertext: Variable length (depends on input)
  • HMAC: 32 bytes (prevents tampering)

Key Derivation (PBKDF2)

  • Algorithm: PBKDF2-HMAC-SHA256
  • Iterations: 100,000
  • Salt: 16 random bytes
  • Output: 64 bytes (32 for encryption + 32 for HMAC)
